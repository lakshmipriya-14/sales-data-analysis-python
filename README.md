# sales-data-analysis-python
Week-2 Internship Project
## Project Overview
This project performs basic sales data analysis using Python. The goal is to calculate total sales, analyze product-wise and category-wise performance, and understand monthly sales trends.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- GitHub

## Dataset Description
The dataset contains 50 sales records with the following columns:
Order_ID, Order_Date, Product, Category, Quantity, Price, Total_Sales, and Month.

## Analysis Performed
- Calculated total sales for each order
- Generated overall sales summary
- Analyzed product-wise sales performance
- Analyzed category-wise sales performance
- Performed monthly sales analysis

## Files in Repository
- Week2_Sales_Data_Analysis.ipynb
- week2_sales_analysis_output.csv

## Conclusion
This project demonstrates how sales data can be analyzed using Python to generate useful business insights.
